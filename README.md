## Design Radix sandbox

Isolated React sandbox that re-implements the `design_radix/*-radix.html` mockups using:
- Radix React primitives (`@radix-ui/react-*`)
- Radix color tokens (`@radix-ui/colors`)

Constraints:
- No color literals in repo files
- No dependency on the real app (`frontend-v2`)

Run:

```bash
npm install
npm run dev
```
